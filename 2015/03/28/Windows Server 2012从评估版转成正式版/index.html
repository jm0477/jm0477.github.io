Windows Server 2012从评估版转成正式版

发表于 2015-03-28

公司某台测试服务器最初安装了Evaluation版Windows Server
2012,测试完成通过KMS激活时提示非核心系统无法激活。这时可以通过将评估版转成正式版后再通过KMS激活。方法如下：

1、以管理员权限运行cmd，输入DISM /online /Get-CurrentEdition，查看当前系统版本，或通过winver命令图形界面查看。

2、输入DISM /online /Set-Edition: /ProductKey:XXXXX-XXXXX-XXXXX-XXXXX-XXXXX
/AcceptEula

为当前可以转换的版本，ProductKey为产品KEY。

3、如需把标准版再升级至数据中心或企业版，可以先输入dism /online /get-
targeteditions，查看当前系统可以升级的目标版本（如下图），再通过第二步命令升级至较高版本。

4、升级完成后，重新启动系统，系统已变成正式版。再通过KMS服务器激活，激活成功。

PS：有关DISM命令的具体用法，请参考http://technet.microsoft.com/zh- cn/library/dd744380(v=ws.10).aspx

DISM /online /Set-Edition:ServerStandard /ProductKey:XC9B7-NBPP2-83J2H-RHMBY-
92BT4 /AcceptEula
