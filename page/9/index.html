使用powershell获取公网ip（附smtp发送邮件脚本）

发表于 2014-06-27

当一个机器连接到互联网中，通常会从ISP（互联网服务提供商）得到一个公网IP地址，但是绝大数机器位于局域网中，因此机器的IP地址和网关的公网IP地址不同。但当机器访问部署在公网上的web服务时，就会把自己的公网IP暴露在外。

PowerShell 3.0 中有一条非常方便的命令Invoke-
WebRequest，可以让用户访问web服务器，并且下载数据，下面的例子通过简单的一行脚本就可以获取用户当前的公网IP。

(``Invoke-WebRequest 'http://myip.dnsomatic.com'
-UseBasicParsing``).Content

另外用户还可以通过 -Proxy 和 –ProxyCredential 指定 代理服务器。

回去在研究定期发送ip到外网的功能，ADSL每次拨号两天就会换一个ip。

今天把smtp发送脚本写好了，下面脚本功能就是：

获取公网ip，通过qq的smtp服务器发出来

定期运行，就可以解决ADSL拨号频繁变更的问题，访问DMZ的主机，暂时找到这么个解决方案。

有同样变态需要的同学，可以copy拿走

$ip=(Invoke-WebRequest ‘http://myip.dnsomatic.com‘ -UseBasicParsing).Content

Write-Host $ip

sendemail($ip)

function sendemail($ReceivedIP)

{

$SmtpServer=”smtp.qq.com”

$smtpuser=”xxx”

$smtppassword=”xxxx”

$mail=New-Object System.Net.Mail.MailMessage

$mailaddress=”xxx@qq.com”

$mailtoAddress=”xxx@qq.com”

$mail.From= New-Object System.Net.Mail.MailAddress($mailaddress)

$mail.To.Add($mailtoAddress)

$mail.Subject=”Schotry Current External IP is $ReceivedIP”

$mail.Priority=”High”

$mail.Body=”the Current IP is $ReceivedIP”

$smtp=New-Object System.Net.Mail.SmtpClient -ArgumentList $SmtpServer

$smtp.Credentials=New-Object System.Net.NetworkCredential -ArgumentList
$smtpuser,$smtppassword

$smtp.Send($mail)

}

快速打开ps1脚本文件3种方法

发表于 2014-06-16

今天有人的OPNVPN连接有问题，后来查看log发现，验证没问题，最后卡在tap 网卡上。。一直不能绑定网卡。

查看网卡状态，果然被禁用了。各种软件都有运行的前提条件，减轻点自己的重复劳动量

索性写了个powershell小脚本，给客户，让他们自己运行一下，脚本完成自动检测问题并自动修复

unblock-file / 可以不更改执行权限的情况下，把本地的ps1文件 获取运行的权限

脚本先不放了，先写上如何快速打开ps1脚本文件吧：

方法1：

为每个 .ps1 配一个 .cmd 批处理文件

这种方法适用于可能需要将 PowerShell
脚本发送给朋友执行，而朋友可能只是初学者或普通用户的场景，并且该脚本不会修改注册表。具体做法是：将以下代码保存为一个 .cmd 文件，放在 .ps1
的同一个目录下。注意主文件名必须和 .ps1 的主文件名一致。

@set Path=%Path%;%SystemRoot%\system32\WindowsPowerShell\v1.0\ & powershell -ExecutionPolicy Unrestricted -NoExit -NoProfile %~dpn0.ps1

例如，您希望朋友执行 My-Script.ps1，那么您只需要将以上代码保存为 My-Script.cmd，放在同一个目录之下发给您的朋友即可

方法2：

用批处理文件做一个简单的菜单，列出 .ps1 文件

该方法由 @PS 网友提供。优点在于可以为多个 .ps1 脚本配一个 .cmd 批处理。执行 .cmd
以后，将显示一个简易的字符界面选择菜单。可以根据用户的选择执行相应的 .ps1 脚本。以下是代码：

@echo off
setlocal enabledelayedexpansionfor  %%i in (*.ps1) do (
     set /a num+=1
     set .!num!=%%i
     echo !num!. %%i)set/p n=这里输入序列：echo !.%n%!set Path=%Path%;%SystemRoot%\system32\WindowsPowerShell\v1.0\ & powershell -ExecutionPolicy Unrestricted -NoProfile .\!.%n%!pause

方法3

使用ftype命令，但是不推荐这种方式，就不列出来

我使用的第一种方法，很适合第一次运行环境使用
