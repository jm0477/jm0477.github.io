用powershell获取文件夹大小

发表于 2014-10-11 |

最近SCOM经常报警 某些服务器C盘磁盘空间太低。

很是烦人，问了快速解决问题。

可以通过删除一些temp file

或者把user的profile删掉，或者挪到其他磁盘。

所以就想到用powershell来快速找到 哪些用户占用了多少空间，然后根据结果，也可以具体问题具体处理

废话少说，上代码：

$directories=Get-ChildItem c:\users -Directory
$ht=@{}
foreach ($directory in $directories){       
[float]$files=Get-ChildItem $directory.FullName -Recurse –File -ErrorAction SilentlyContinue |Measure-Object -sum length| select Sum -ExpandProperty sum
Write-Host foleder $directory size is ($files)
}

后续会整合成一个fuction，按空间大小排序， 可以调用输出成报表，敬请期待吧。
