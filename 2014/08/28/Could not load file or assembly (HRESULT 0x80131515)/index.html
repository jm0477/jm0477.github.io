Could not load file or assembly (HRESULT 0x80131515)

发表于 2014-08-28 |

下载的附件里面包含dll文件，会自动被系统block，无法使用

必须手动点开属性，点unblock

彻底解决这个问题，去组策略里面启动一个：

  1. 1.gpedit.msc

  2. 2. User Configuration > Administrative Templates > Windows Components > Attachment Manager

  3. 3. Enable: Do not preserve zone information in file attachments

再下载的附件，就不会被系统自动锁住，导致标题这个错误了。
