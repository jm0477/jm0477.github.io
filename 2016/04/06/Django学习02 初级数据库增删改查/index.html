在views.py中调用方法：

from django.http import HttpResponse

from django.template import loader, Context

from myblog.models import Student

def student_list(request):

t = loader.get_template(“student_list.html”)

studentList = Student.objects.filter( age__gt = 16)

c = Context({“studentList”:studentList})

return HttpResponse(t.render(c))

__exact 精确等于 like ‘aaa’

__iexact 精确等于 忽略大小写

__contains 包含 like ‘%aaa%’

icontains 包含 忽略大小写，但是对于sqlite来说，contains与__icontains作用相同

__gt 大于

__gte 大于等于

__lt 小于

__lte 小于等于

__in 存在于一个list范围内

__startswith 以..开头

__istartswith 以..开头忽略大小写

__endswith

__iendswith

__range 在..范围内

__year 日期字段的年份

__month 日期字段的月份

__day 日期字段的日

__isnull=True/False 是否为空
