Django学习07 django通用视图- 基本概念及写法

发表于 2016-04-08

urls.py

通用视图的包：

from django.views.generic.simple import direct_to_template

urlpatterns = patterns(‘’,

(r’^about/$, direct_to_template, {‘template’: ‘about.html’})

)

当请求about的视图时，就是直接调用模板页面about.html

=======================================================================================================

urlpatterns = patterns(‘’,

(r’^about/$, direct_to_template, {‘template’: ‘about.html’})

(r’^about/(\w+)/$’, abouts)

)

views.py:

from django.template import TemplateDoesNotExisted #模板不存在

from django.template import Http404

from django.views.generic.simple import direct_to_template

def abouts(request, page):

try:

return direct_to_template(request, template = ‘about/$s.html’, % page)

except TemplateDoesNotExisted:

raise Http404()

功能是，接收到传入的url及页码，如果存在，直接跳转相应的template页面，如果不存在template，那返回404页面

=========================================================================================================

针对publisher对象做一个通用视图

from django.views.generic import list_detail

publisher_info = {

‘queryset’: Publisher.objects.all()

#lesson1/publisher_info.html 如果没指明，会自动寻找此模板

‘template’: ‘publisher_info.html’

}

urlpatterns = patterns(‘’,

(r’^publisher/$’, list_detail.object_list, {‘publisher_info’: publisher_info})

)

功能是，
接受publisher/请求时，将Publisher对象的一个queryset内容传递给指定的模板publisher_info.html(这个模板要在Publisher_info字典里的’template’字段定义)
