Windows10配置ssh-key免密登录github

发表于 2017-10-17

先说下配置的需求，我这博客是用hexo框架+github.io做静态展示
所以在hexo部署的时候需要hexo deploy一键来部署到github上，linux下很简单，windows下也类似。下面记录下折腾的步骤

  1. 打开Git Bash命令行窗口
    ls -al ~/.ssh 查看下是否有已经存在的证书，如有请删掉，如没有继续
  2. 输入如下指令(邮箱是你github的帐号):
    ssh-keygen -t rsa -C "richie.jing@msn.cn"
    回车，回车

  3. 输入如下命令：
    evalssh-agent -s
    ssd-add`

  4. 复制刚刚生成的key到粘贴板
    ‘clip < ~/.ssh/id_rsa.pub’

  5. 网页打开github的ssh key设置，右键添加一个新的sshkey上去即可

#测试效果
键入以下命令看你的key是否生效：
ssh -T git@github.com

如果提示让你输入yes/no基本就ok了

完成部署，最后一步：
hexo generate
hexo deploy

搞定～
